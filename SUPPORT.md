# Support 🙋

HA TV PiP is early-stage software.

## Getting Help 🧭

For now, use GitHub issues for:

- Android TV build problems.
- PiP behavior bugs.
- Test stream playback problems.
- Discovery or pairing problems.
- Documentation improvements.

Please include:

- Device or emulator model.
- Android version.
- App version or commit.
- Steps to reproduce.
- Relevant logs, if available.

## Not Supported Yet 🚧

These features are planned for later and are not supported yet:

- Camera entity service calls.
- Snapshot popups.
- WebRTC.
- Remote access.
